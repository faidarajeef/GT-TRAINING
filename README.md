#  Zomato Restaurant Data Analysis

##  Project Overview
This project focuses on analyzing Zomato restaurant data to understand the key factors that influence restaurant success.  
Using Python and statistical techniques, the project explores how cost, ratings, votes, cuisines, and online delivery impact restaurant performance and customer perception.

---

##  Tools & Technologies
- Python  
- Jupyter Notebook (Anaconda)  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  

---

##  Dataset
- Source: Zomato restaurant dataset (provided in Excel/CSV format)
- Domain: Food Technology & Hospitality
- Data includes:
  - Restaurant ratings
  - Average cost for two
  - Votes (reviews)
  - Cuisines
  - City-wise information
  - Online delivery availability

---

##  Project Phases

###  Phase 1: Data I/O & Initial Exploration
- Loaded and inspected the dataset
- Identified numerical and categorical features
- Performed descriptive statistics on restaurant pricing
- Analyzed top cuisines by frequency

---

### Phase 2: Data Cleaning & Feature Engineering
- Handled missing values in ratings using city-wise averages
- Encoded categorical Yes/No features into binary values
- Normalized cost data for fair comparison

---

### Phase 3: Statistical Analysis
- Calculated correlation between votes and ratings
- Computed conditional probability of high ratings based on online delivery
- Visualized rating distribution to study rating patterns

---

### Phase 4: Exploratory Data Analysis (EDA)
- Visualized cost vs rating relationships
- Compared ratings across different cuisines using boxplots
- Analyzed city-wise value-for-money trends

---

### Phase 5: Business Insights & Recommendations
- Identified high-demand cuisines with rating gaps
- Suggested optimal cities and cuisines for new restaurant owners
- Provided data-driven strategic recommendations

---

## Key Insights
- Online delivery shows a positive influence on restaurant ratings
- Popular cuisines do not always guarantee higher ratings
- Certain cities offer better value-for-money dining options

---

##  Conclusion
This project demonstrates how data analysis and visualization can be used to extract meaningful insights from real-world datasets and support informed business decisions in the food and hospitality industry.
